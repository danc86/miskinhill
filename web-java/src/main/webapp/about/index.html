About

Miskin Hill Academic Publishing (ABN 27 712 504 809) is a Brisbane-based business, providing electronic publishing and editing services to academic journals. To find out more about what we can do for your academic publication, please get in touch with us.

Areas of expertise

  • Typesetting and preparation for print, using LaTeX, Adobe InDesign, or Microsoft Word
  • Copyediting services with a meticulous attention to detail
  • Migration between document formats
  • Deployment of cutting-edge web technologies for online publications, including:
    • OpenURL, for connecting users to cited works in their local library
    • OAI-PMH, for content discovery by aggregators
    • RDF, for machine-readable metadata on the Semantic Web
  • Good web citizenship:
    • persistent, memorable URLs
    • clean, semantic markup adhering to web standards
    • interoperability across browsers, devices, and platforms
